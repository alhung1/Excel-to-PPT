Test Purpose & Scope

This report validates the RBE773v2 PR Sample against the NRH Beta Sample baseline across two test domains: Backhaul throughput (BASE-LAN to Satellite-LAN, MLO 5+6GHz) and Fronthaul RvR (Mesh system throughput vs. range with Intel BE200 WiFi-7 client using Auto Channel roaming across 8 test locations). The goal is to confirm no regression exists between the PR and Beta hardware stages before production release.

  • 1
    Backhaul Performance: DL and UL throughput over BASE–SAT1 and BASE–SAT2 links using MLO (5GHz CH44 EHT160 + 6GHz CH69 HT320) in STAR topology.
  • 2
    Fronthaul Mesh RvR: UL+DL, DL-only, and UL-only client throughput at 8 real-world locations ranging from 10ft LOS to 110ft NLOS, with automatic channel and AP association.
  • 3
    Auto-Channel Matrix: Verification that both samples select equivalent channels and AP associations at each test location under identical conditions.

Key Findings & Conclusions
  • 1
    Backhaul — PR Sample slightly leads: On BASE–SAT1, PR Sample achieves 2,273 Mbps DL (+0.9%) and 2,289 Mbps UL (+1.2%) over Beta. On BASE–SAT2, PR leads by 4.6% DL and 4.2% UL. All differences are within normal variation range.
  • 2
    Fronthaul DL — Beta Sample marginally leads at most locations: Beta achieves ~0.6% higher average DL throughput (800 vs 795 Mbps). Both devices deliver comparable peak performance at LOS locations (L1: PR 1983 vs Beta 2012 Mbps).
  • 3
    Fronthaul UL — Beta Sample marginally leads: Beta averages ~1.3% higher UL throughput (762 vs 752 Mbps). Performance gap is smallest at far locations (L8: PR 27.1 vs Beta 29.1 Mbps).
  • 4
    Auto Channel — Identical selection across all locations: Both PR and Beta samples select CH69 6GHz for L1–L7 and fall back to CH44 5GHz only at L8 (110ft, 103 dB path loss). AP association (Base → SAT1 → SAT2) is consistent between devices.
  • 5
    Overall Verdict — PASS: All performance differences are within ±5% tolerance. No regressions observed between PR and NRH Beta hardware stages. The RBE773v2 PR Sample is production-ready relative to the Beta baseline.
